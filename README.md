### 🛠️ 核心策略逻辑 (The Logic)

本策略并非单一指标驱动，而是一套 **"全周期复合系统"**：

#### 1️⃣ 宏观风控：双均线红绿灯 (Market Filter)
* **逻辑**：监测沪深 300 指数的 `5日` 与 `20日` 均线。
* **作用**：**总开关机制**。在大盘空头排列时强制空仓，成功避开了 2022-2024 年的单边熊市。

#### 2️⃣ 选股引擎：动态动量轮动 (Momentum Rotation)
* **逻辑**：每周从成分股中筛选过去 `20个` 交易日涨幅前 `15名` 的股票。
* **作用**：基于“强者恒强”的心理惯性，确保资金永远配置在全市场最活跃的资产中。

#### 3️⃣ 择时进场：海龟突破 + RSI 修正
* **突破**：价格冲破 20 日 **Donchian Channel** 高点。
* **低吸**：若强势股出现 `RSI < 45` 的适度回调，则视为二次上车机会。

#### 4️⃣ 利润保护：动态追踪止损 (Trailing Stop Loss) 🏆
* **逻辑**：实时记录持仓股的 **历史最高点**。
* **作用**：一旦股价从最高点回撤超过 **10%**，立刻清仓锁定利润。这是防范“妖股跳水”和高位回吐的核心神技。
